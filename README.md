# Automatic Testing Practice

This repository contains a QA automation practice project created as part of my QA learning path.

## Project Overview

The project demonstrates basic automated testing skills using Cypress, GitHub Actions, and a Gherkin test scenario.

## Technologies and Tools

- Cypress
- JavaScript
- GitHub Actions
- Gherkin
- Git / GitHub

## What was practiced

- Creating automated test scenarios
- Working with Cypress project structure
- Running tests locally
- Using GitHub Actions for test execution
- Writing a Gherkin-style purchase scenario
